# MCP2API Bridge (浏览器插件)

[🇺🇸 English](README.md)

这是一个 Chrome/Edge 浏览器扩展，用于将网页版 AI (ChatGPT/Gemini) 连接到你的本地 MCP 网关，从而让 AI 安全地操作本地工具。

## ✨ 功能特性
- **安全隐私**: 基于 Manifest V3 架构，所有请求通过后台服务转发，符合浏览器安全规范。
- **即时配置**: 点击插件图标即可弹出配置面板，支持修改 API 端口和切换“自动发送”模式。
- **视觉反馈**: 自动识别 MCP JSON 指令并添加绿色边框高亮，状态一目了然。

## 📥 安装指南

1.  打开 Chrome 或 Edge 浏览器，在地址栏输入 `chrome://extensions`。
2.  开启右上角的 **“开发者模式” (Developer mode)** 开关。
3.  点击左上角的 **“加载已解压的扩展程序” (Load unpacked)** 按钮。
4.  选择本项目中的 `Extension` 文件夹。

## ⚙️ 配置说明

点击浏览器工具栏上的插件图标即可打开设置面板：
- **Gateway Port (网关端口)**: 默认为 `3000`。如果你修改了本地网关的端口，请在此同步修改。
- **Auto Send (自动发送)**:
    - ✅ **开启**: 插件回填执行结果后，会自动按下回车发送给 AI。
    - ❌ **关闭**: 插件仅回填结果，你需要人工检查后再手动发送。

## 🚀 使用方法

1.  确保本地 MCP 网关正在运行 (在项目根目录执行 `npm start`)。
2.  打开 ChatGPT 或 Google Gemini 网页。
3.  按下 F12 打开控制台，若看到 `[MCP Extension] ...` 字样说明插件已激活。
