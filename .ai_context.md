## 1. Project Overview (WebMCP)

**Goal**: Connect Web-based AI (ChatGPT/DeepSeek) to Local VS Code via MCP Protocol.

### 🏗 Structure
- **Root**: Contains build scripts (`build_release.sh`) and global docs.
- **`/mcp-gateway-vscode`** (VS Code Extension):
  - Acts as the MCP Server (Express).
  - **Port**: Default `34567` (Dynamic range 30000-40000).
  - **Key File**: `src/gateway.ts` (Handles `/bridge` page and WebSocket/HTTP).
  - **Logic**: Generates a secure Token, requires `WebMCP Bridge` to function.
- **`/mcp-bridge-browser`** (Browser Extension):
  - Injects into AI pages to intercept tool calls.
  - **Key Files**:
    - `content.js`: Intercepts JSON tool calls, handles Auto-Prompt, Periodic Training, and **i18n Logging**.
    - `background.js`: Persistent Service Worker. Handles connection and **preloads i18n resources**.
    - `popup.html/js`: Connection status & "Copy System Prompt".
  - **i18n Resources** (Preloaded by background.js):
    - `prompt.md` / `prompt_zh.md`: System prompts.
    - `error_hint.md` / `error_hint_zh.md`: Strict format error warnings.
    - `train.md` / `train_zh.md`: Gentle reminders for periodic training.

## 2. Key Mechanisms (Do Not Break)

### 🔄 Cross-Promotion (互推逻辑)
- **Gateway (VS Code)**: The `/bridge` page (in `gateway.ts`) checks `document.documentElement.getAttribute('data-extension-installed')`. If missing, shows a red warning to install the browser extension.
- **Bridge (Browser)**: The Popup shows distinct guides for "Already Installed" (Start Server) vs "Not Installed" (Go to Marketplace).

### 🧠 AI Memory Maintenance
- **Periodic Training**: `content.js` counts tool calls. Every 5th call, it injects a `system_note`.
  - **Logic**: Uses `train.md` (or `_zh`) for gentle reinforcement, distinct from error handling.
  - **i18n**: Logic in `content.js` selects language based on `navigator.language`.

## 5. Implementation Notes (Critical Logic)

### 🔧 Browser Bridge (`content.js`)
- **Input Backfill Logic**: We MUST use `document.execCommand('insertText')` instead of direct `innerText` assignment.
  - **Reason**: `contenteditable` divs (like in ChatGPT/DeepSeek) render `\n` as new paragraphs `<p>`, causing massive empty gaps (Double Spacing) if assigned directly.
  - **Flow**: Compress History (`\n+` → `\n`) → Join with `\n\n` → `execCommand` (Simulates user typing).

### 🛠 VS Code Gateway
- **Debugging**: Added `$(output) View Logs` to the Status Bar Menu (Quick Pick). Opens the "MCP Gateway" output channel directly.

### 📦 Build Scripts
- **Windows PowerShell**: Avoid Emojis (🚀, ✅). Use ASCII tags (`[OK]`, `[ERROR]`) to prevent encoding (Mojibake) issues on default Windows terminals.
