# 快速上手

本指南将帮助你快速了解如何使用 WebMCP。

## 环境准备

在开始之前，请确保你的环境中已安装：

- [Node.js](https://nodejs.org/) (建议 v18+)
- [pnpm](https://pnpm.io/) (建议 v8+)

## 安装方式

### 1. VSCode 插件
在 VSCode 扩展市场搜索 **WebMCP** 即可一键安装。

### 2. 桌面客户端
前往 [GitHub Releases](https://github.com/three-water666/WebMCP/releases) 页面，根据你的操作系统下载对应的安装包：
- Windows: `.exe`
- macOS: `.dmg` (Intel/Apple Silicon)
- Linux: `.AppImage`

## 基本配置

1. 启动 WebMCP 客户端或插件。
2. 在设置面板中配置你的 MCP 服务地址。
3. 确保你的 AI 客户端（如 Claude Desktop）已连接到 WebMCP 网关。

## 常见问题

**Q: 为什么无法连接到 MCP 服务？**
A: 请检查网络连接以及 MCP 服务是否已正常启动。

**Q: 支持哪些 AI 模型？**
A: WebMCP 作为网关，理论上支持所有兼容 MCP 协议的模型和客户端。
